This folder contains three datasets for NDV phylogenetic reconstruction:

Class I full fusion - 619 sequences (303 new sequences)
Class II full fusion - 2157 sequences (256 new sequences)
Class II pilot dataset - 128 sequences representing all class II NDV genotypes and sub-genotypes (3 new sequences)
Class I complete genome - 177 sequences (31 new sequences)
Class II complete genome - 691 sequences (80 new sequences)

These are updated sequence datasets of the original datasets in the publication cited above and include all sequences 
available in GenBank as of May 9th 2022 after curation (removing duplicates, vaccines, clones, recombinant forms, etc.).
